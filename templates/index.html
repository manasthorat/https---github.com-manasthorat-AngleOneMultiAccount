Getting Started

Welcome to the Angle One Multi-Account Trading System. This system allows you to automate trading across multiple Angle One accounts using TradingView webhooks.

To get started:

1

Configure Accounts

Set up your Angle One accounts in the Accounts section

2

Configure Alerts

Set up TradingView alerts to send webhooks to your server

3

Monitor Trading

Track your orders and positions in the dashboard

Key Features
Multi-Account Management

Manage multiple Angle One trading accounts from a single interface.

Advanced Position Tracking

Monitor your positions with real-time P&L updates and performance metrics.

Automated Trading

Execute trades automatically based on TradingView alerts and custom strategies.
